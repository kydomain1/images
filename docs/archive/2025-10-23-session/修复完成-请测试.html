🔧 JavaScript 错误已修复

❌ 问题原因

重复定义变量：

// 第 67 行
const bgAdvancedPanel = document.getElementById('bg-advanced-panel');

// 第 138 行（重复！）
const bgAdvancedPanel = document.getElementById('bg-advanced-panel');

这会导致 JavaScript 报错，页面功能无法正常工作。

✅ 解决方案

已删除重复的变量定义，使用已声明的变量。

// 只保留一次定义
const bgAdvancedPanel = document.getElementById('bg-advanced-panel');

// 后续直接使用
if (toggleBgAdvanced && bgAdvancedPanel) { ... }
