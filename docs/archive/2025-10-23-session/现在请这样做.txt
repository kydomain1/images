问题：服务器无法启动（端口被占用）

解决：需要手动操作一下


═══════════════════════════════════════════════════════════

                    👉 立即执行：

═══════════════════════════════════════════════════════════


步骤1：重启电脑（最简单最有效）

  这会清除所有占用端口的进程


步骤2：打开文件夹

  D:\images


步骤3：双击运行

  启动服务器-最终版.bat


步骤4：等待看到

  "✅ 服务器运行在 http://localhost:3000"


步骤5：打开浏览器访问

  http://localhost:3000/tool.html


═══════════════════════════════════════════════════════════

            或者不重启电脑，手动执行：

═══════════════════════════════════════════════════════════


方法A：使用任务管理器

  1. 按 Ctrl+Shift+Esc 打开任务管理器
  2. 找到所有 "Node.js" 进程
  3. 右键 → 结束任务
  4. 双击运行：启动服务器-最终版.bat


方法B：使用命令行

  1. 打开新的 CMD 窗口（Win+R，输入cmd）
  2. 输入：taskkill /F /IM node.exe
  3. 输入：cd /d D:\images
  4. 输入：node server-with-r2.js


═══════════════════════════════════════════════════════════

              ✅ 成功后应该看到：

═══════════════════════════════════════════════════════════

服务器窗口：
  ✅ 服务器运行在 http://localhost:3000
  ✅ 通义万相代理服务器运行中...

浏览器访问 http://localhost:3000/tool.html：
  ✅ 网站正常打开
  ✅ 可以切换到"图片编辑器"
  ✅ 可以上传图片

测试背景移除：
  ✅ 上传图片
  ✅ 点击"移除背景"
  ✅ 按F12查看控制台
  ✅ 显示："Remove.bg API 成功！"


═══════════════════════════════════════════════════════════

            💡 推荐方案（最简单）：

═══════════════════════════════════════════════════════════

  1. 重启电脑（30秒）
  2. 双击：启动服务器-最终版.bat
  3. 打开浏览器访问：http://localhost:3000/tool.html
  4. 开始使用！
