## 📝 使用说明

### 对于开发者
1. 所有翻译存储在 `js/i18n.js`
2. 使用 `data-i18n` 属性绑定翻译
3. 表单占位符使用 `data-i18n-placeholder`
4. 语言选择自动保存在 `localStorage`

### 对于用户
1. 点击页面右上角语言选择器
2. 选择您的首选语言
3. 所有页面将自动切换
4. 下次访问时保持您的选择
