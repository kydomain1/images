🔍 使用方法
==============================================

用户操作步骤:
1. 打开任意政策页面
   - http://localhost:3000/pages/privacy.html
   - http://localhost:3000/pages/terms.html
   - http://localhost:3000/pages/about.html
   - http://localhost:3000/pages/contact.html

2. 点击右上角的语言按钮 (🌐 中文 ▼)

3. 从下拉菜单中选择目标语言

4. 页面内容立即切换为选定语言

5. 刷新页面，语言设置会自动保持

localStorage.setItem('preferredLanguage', lang);

==============================================
⚠️ 注意事项
==============================================

1. 清除浏览器缓存
   - 首次使用时建议清除缓存
   - 或使用 Ctrl+Shift+R 硬刷新

2. 本地存储
   - 语言偏好保存在 localStorage
   - 清除浏览器数据会重置为默认语言

3. 脚本加载顺序
   - i18n.js 必须在 policy.js 之前加载
   - 当前顺序正确，无需调整

4. 浏览器兼容性
   - 支持所有现代浏览器
   - IE11可能需要polyfill
