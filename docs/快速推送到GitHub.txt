========================================
  AI图片生成器 - GitHub 推送指南
========================================

📦 目标仓库: https://github.com/kydomain1/images.git

========================================
  方法 1: 双击运行脚本（最简单）
========================================

1. 找到文件: push-to-github.bat
2. 双击运行
3. 等待完成
4. 访问: https://github.com/kydomain1/images

========================================
  方法 2: 手动执行命令
========================================

打开 PowerShell 或 CMD，在项目文件夹中执行：

git init
git add .
git commit -m "Initial commit: AI图片生成器完整项目"
git branch -M main
git remote add origin https://github.com/kydomain1/images.git
git push -u origin main

========================================
  如果遇到问题
========================================

1. 提示 "remote origin already exists"
   解决: git remote remove origin
   然后重新执行 git remote add origin 命令

2. 需要输入密码
   - 使用 GitHub Personal Access Token
   - 不要使用账户密码

3. 推送被拒绝 (rejected)
   - 检查远程仓库是否为空
   - 如需强制推送: git push -u origin main --force
     (⚠️ 注意：会覆盖远程内容)

========================================
  推送的内容
========================================

✅ 包含:
  - 所有 HTML、CSS、JS 文件
  - 图片和资源文件
  - 文档和说明
  - 配置文件 (package.json, sitemap.xml 等)

❌ 排除 (已在 .gitignore):
  - node_modules/
  - .env (API 密钥)
  - uploads/ (上传的文件)

========================================
  推送后的步骤
========================================

1. 访问 GitHub 仓库验证
2. 在 Settings → Secrets 中添加环境变量:
   - TONGYI_API_KEY
   - R2_ACCESS_KEY_ID
   - R2_SECRET_ACCESS_KEY
   - R2_BUCKET_NAME
   - R2_ENDPOINT
   - R2_PUBLIC_URL

3. (可选) 部署到 Vercel 或 Netlify
