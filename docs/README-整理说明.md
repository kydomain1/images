## 🛠️ 维护指南

### 添加新文档时

1. **命名规范：**
   - 功能说明：`功能名称-说明.md`
   - 配置指南：`服务名称-配置指南.md`
   - 完成报告：`任务名称-完成报告-日期.md`
   - 检查报告：`检查类型-报告-日期.md`

2. **格式选择：**
   - 优先使用 `.md` 格式
   - 仅快速笔记使用 `.txt`
   - 英文文档：`UPPERCASE_UNDERSCORE.md`
   - 中文文档：`连字符-命名.md`

3. **更新本说明文档**

### 定期整理

**每月执行一次：**
```bash
scripts\整理文件夹.bat
```

这会自动将已完成的任务文档移至 `archive/`
