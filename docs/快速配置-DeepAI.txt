========================================
DeepAI API - 快速配置指南
========================================

📌 第一步：获取API Key
----------------------------------------
1. 访问：https://deepai.org/
2. 点击右上角 "Sign Up" 注册账号
   - 使用邮箱注册（支持Gmail、QQ邮箱等）
   - 验证邮箱
   - 无需信用卡，无需实名

3. 登录后访问：https://deepai.org/dashboard/profile
4. 找到 "API Key" 部分
5. 点击 "Copy" 复制API Key

💡 新用户福利：
   - 每月免费调用次数
   - 立即可用，无需等待
   - 支持多种AI模型


📌 第二步：配置API Key
----------------------------------------
打开文件：js/api-config.js

找到第2行，修改为：
    provider: 'deepai',

找到第22-25行，修改为：
    deepai: {
        apiKey: '你的DeepAI-API-KEY',  ← 粘贴你的API Key
        endpoint: 'https://api.deepai.org/api/text2img'
    }

保存文件！


📌 第三步：测试
----------------------------------------
方法1：使用测试工具（推荐）
   1. 双击打开：test-deepai.html
   2. 输入API Key
   3. 选择模型（默认"标准模型"即可）
   4. 输入提示词（建议用英文）：
      "a cute cat sitting on a windowsill"
   5. 点击"开始生成"

方法2：使用主工具
   1. 打开：tool.html
   2. 输入提示词（中英文都支持）
   3. 点击"生成图片"
   4. 等待3-10秒


📌 支持的模型
----------------------------------------
✅ Text2img（标准模型）
   - 通用图片生成
   - endpoint: text2img

✅ Stable Diffusion
   - 更高质量
   - endpoint: stable-diffusion

✅ Cute Creature（可爱生物）
   - 专门生成可爱角色
   - endpoint: cute-creature-generator

✅ Fantasy World（奇幻世界）
   - 生成奇幻场景
   - endpoint: fantasy-world-generator

✅ Cyberpunk（赛博朋克）
   - 赛博朋克风格
   - endpoint: cyberpunk-generator

💡 切换模型：修改 js/api-config.js 中的 endpoint


📌 提示词技巧
----------------------------------------
✅ 建议使用英文（效果最好）：
   "a cute orange cat sitting on a windowsill, 
    sunlight shining on it, oil painting style, 
    highly detailed, warm colors, soft lighting"

✅ 风格关键词：
   - oil painting (油画)
   - digital art (数字艺术)
   - photorealistic (逼真照片)
   - anime style (动漫风格)
   - watercolor (水彩)
   - 3D render (3D渲染)
   - cyberpunk (赛博朋克)

✅ 质量关键词：
   - highly detailed (高细节)
   - 4k resolution (4K分辨率)
   - masterpiece (杰作)
   - best quality (最佳质量)

❌ 不推荐：
   - 过于简单的中文："猫"
   - 没有细节描述


📌 常见问题
----------------------------------------
Q: 提示"Invalid API Key"
A: 检查API Key是否正确复制，确保没有多余空格

Q: 提示"Rate Limit Exceeded"
A: 超过速率限制，等待1分钟后重试
   免费版限制：约5-10次/分钟

Q: 提示"Insufficient Credits"
A: 免费额度用完，可以：
   1. 等待下月重置
   2. 升级到付费计划（$4.99/月起）
   3. 切换到其他免费API（如Pollinations）

Q: 图片有水印
A: 免费版会有DeepAI水印
   解决：升级到付费计划去除水印

Q: 生成速度慢
A: 正常情况3-10秒
   如果更慢，可能是：
   - 网络问题
   - API服务器繁忙
   - 建议换个时间重试
